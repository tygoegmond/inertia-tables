## What is Inertia Tables?

Inertia Tables is a powerful Laravel package that provides seamless integration between your Laravel backend and React frontend for building interactive data tables. Built specifically for Inertia.js applications, it offers a complete solution for displaying, searching, sorting, and managing tabular data with minimal configuration.

## Quick Start

### 1. Install the Package

```bash
# Install the Laravel package
composer require tygoegmond/inertia-tables

# Install the React components
npm install @tygoegmond/inertia-tables-react
```

### 2. Create a Table Class

```php
<?php

namespace App\Tables;

use App\Models\User;
use Egmond\InertiaTables\Concerns\InteractsWithTable;
use Egmond\InertiaTables\Contracts\HasTable;
use Egmond\InertiaTables\Table;
use Egmond\InertiaTables\Columns\TextColumn;
use Egmond\InertiaTables\Actions\Action;

class UserTable implements HasTable
{
    use InteractsWithTable;

    public function table(Table $table): Table
    {
        return $table->as('users')
            ->query(User::query())
            ->columns([
                TextColumn::make('name')->searchable()->sortable(),
                TextColumn::make('email')->searchable()->sortable(),
                TextColumn::make('created_at')->sortable(),
            ])
            ->actions([
                Action::make('edit')->label('Edit User'),
                Action::make('delete')->label('Delete')->color('danger'),
            ])
            ->searchable()
            ->paginate(15);
    }
}
```

### 3. Use in Your Controller

```php
<?php

namespace App\Http\Controllers;

use App\Tables\UserTable;
use Inertia\Inertia;

class DashboardController extends Controller
{
    public function index()
    {
        return Inertia::render('Dashboard', [
            'users' => UserTable::make(),
        ]);
    }
}
```

### 4. Render in React

```tsx
import { InertiaTable } from '@tygoegmond/inertia-tables-react';

export default function Dashboard({ users }) {
  return (
    <div className="p-6">
      <h1 className="text-2xl font-bold mb-6">Users</h1>
      <InertiaTable state={users} />
    </div>
  );
}
```

## Core Concepts

### Tables
The foundation of the package. Tables define the structure, data source, and behavior of your data display.

### Columns
Flexible column types for displaying different kinds of data, from simple text to complex formatted values with badges, prefixes, and custom formatting.

### Actions
Interactive operations that users can perform on your data:
- **Row Actions**: Operations on individual records (edit, delete, view)
- **Bulk Actions**: Operations on multiple selected records
- **Header Actions**: Table-level operations (create, export, import)

### Search & Filtering
Built-in search functionality with support for column-specific searching, relationship searching, and custom search logic.

### Sorting & Pagination
Efficient sorting and pagination with support for multiple columns, relationship sorting, and custom pagination logic.
