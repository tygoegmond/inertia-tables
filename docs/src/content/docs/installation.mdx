## Requirements

Inertia Tables requires the following to run:

- PHP 8.2+
- Laravel v12.0+
- Tailwind v4.0+

## Installation

Require the package using Composer:

```bash
composer require tygoegmond/inertia-tables
```

### NPM Installation
After installing the package you will need to install the required npm package.

```bash
npm install @tygoegmond/inertia-tables
```

### Configuring styles

Use the `@source` directive to import the Inertia Tables styles in your `app.css` file:

```css
@source '../../vendor/tygoegmond/inertia-tables/react/src/**/*.{js,jsx}';
```

### Compiling assets

Compile your assets using `npm run dev`.
